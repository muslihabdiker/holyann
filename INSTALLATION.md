# Installation Guide

## Prerequisites
- Node.js v16 or higher
- PostgreSQL 14 or higher

## Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/user/repo.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure the database:
   ```bash
   cp .env.example .env
   ```
4. Start the server:
   ```bash
   npm start
   ```
