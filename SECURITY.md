# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability, please contact us at [security@example.com]. 
We will respond within 48 hours and keep you updated on progress.

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | ✅                 |
| 0.x.x   | ❌                 |
